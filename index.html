youngStar

youngStar is a website for youngsters to visualize schoolwork successes and failures and be rewarded for their successes. The goal of youngStar is for children in elementary and middle school to learn early in life that hard work and good grades will be rewarded.

youngStar was my final project for the coding bootcamp I was attending. In a 2 week time span I researched and built this application using React.js, Backbone.js, Bootstrap, Parse and jQuery. I used 2 data models and trello for the user stories.

The design for this project was done by a very talented designer that I use to work with named Eric Malcom. Please check out his website here: ericmalcolm.com
